### Fix Patterns (cycle 12)
- **AC 由来の grep をそのまま test doc に転記する罠**: Issue の AC-1 が固定文字列 grep (`grep -rnF ':completed]'`) を使っていても、その literal が捕捉できる sentinel 形式は限定的 (末尾 `:completed]` の形式のみ)。suffix を持つ sentinel (`[create:completed:{N}]` / `[lint:completed:auto]`) を取り逃すため、残存チェックは suffix 許容 regex (`\[[a-z]+:completed(:[^]]*)?\]`) を使う。AC の literal を盲目的に転記せず、実際に捕捉すべき形式を網羅できるか検証する。
