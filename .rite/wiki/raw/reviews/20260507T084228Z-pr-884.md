### Finding Patterns

主要な検出パターン:
1. **Bash hook の trap ERR + set -euo pipefail interaction**: `trap 'exit 0' ERR` が intentional `exit 2` の手前にある外部コマンド (jq / awk) の失敗を silent に握りつぶす silent-failure 経路。STRICT mode で BLOCK 契約を破る
